# ArtInChip D13x GE/DE Feature Tracking (机能追踪)

> **Status**: 1.3.0
> **Target**: D133CCS

## 1. Graphics Engine (GE) - 2D 图形加速引擎

| 机能分类 | 具体功能 | 状态 | 关联 Demo / 备注 |
| :--- | :--- | :--- | :--- |
| **BitBLT** | 基础位块搬移 | **已使用** | 全特效通用管线基础 |
| **Scaler** | 硬件缩放 (Bilinear) | **已使用** | 全屏覆盖的核心保证，Night 50 验证了开环扩张 |
| **Rot0** | 90/180/270 度固定旋转 | **未使用** | 优先级：低 |
| **Rot1** | **任意角度旋转** | **已使用** | Night 21/22/48/49 的灵魂机能 |
| **Blending** | Porter-Duff 混合 (Rule 11 ADD) | **已使用** | Night 22/47 验证了高能干涉，注意 0=Enable |
| **Color Key** | 特定颜色透明化 | **已使用** | Night 29/41 验证了镂空叠加 |
| **Fill** | 单色/渐变矩形填充 | **已使用** | 关键：用于缓冲区真空化、反物质擦除 |
| **Dither** | 硬件抖动 (高位转低位) | **未使用** | 潜力点：优化 RGB565 色阶平滑度 |
| **Flip** | **水平/垂直镜像** | **已使用** | Night 26/36/44 对称美学核心 |
| **CMDQ** | 命令队列异步提交 | **部分使用** | 尚未实现极致的 Batch 级合并 |

## 2. Display Engine (DE) - 显示输出引擎

| 机能分类 | 具体功能 | 状态 | 关联 Demo / 备注 |
| :--- | :--- | :--- | :--- |
| **UI Layer** | UI 图层 (Layer 1) | **已使用** | 核心渲染层 |
| **VI Layer** | **视频图层 (Layer 0)** | **尝试后回退** | 尝试 YUV 时由于 GE 格式限制回退 |
| **Multi-Win** | UI 层 4 窗口模式 | **已使用** | Night 25 超子镜像核心 |
| **YUV Support**| YUV420/422/444 支持 | **尝试后回退** | GE BitBLT 不支持 YUV420P 源输入，仅支持 YUV400 |
| **CSC** | 颜色空间转换 (YUV->RGB) | **已使用** | 硬件底层静默运行 |
| **CCM** | **颜色校正矩阵 (滤镜)** | **已使用** | Night 24/31/50 实现了全谱色彩偏移 |
| **Gamma** | Gamma 矫正查找表 | **已使用** | Night 30/35 验证了非线性亮度映射 |
| **HSBC** | **亮/对比/饱和/色调调节** | **已使用** | Night 26/43 验证了全局画质干涉 |

## 3. Memory & Bus (总线与内存)

| 机能分类 | 具体功能 | 状态 | 备注 |
| :--- | :--- | :--- | :--- |
